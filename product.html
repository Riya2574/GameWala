Connecting and Installing

Connect your PS4™ system to your TV. Make all connections before plugging the AC power cord into an electrical outlet.

  • AAC IN connector (PS4™ system)
  • B HDMI OUT port (PS4™ system)
  • C HDMI input port (TV)
  • D HDMI cable
  • E AC power cord
  • F To an electrical outlet

When using the PS4™ system vertically, attach the vertical stand (sold separately).

You can connect the system to the Internet using Wi-Fi® or a LAN (Ethernet) cable. Make sure you have a LAN cable if you don’t plan to use Wi-Fi.

Connection Example

The following illustration shows an example of a wired network connection for your PS4™ system.

  • A LAN port (PS4™ system)
  • B LAN cable
  • C Router
  • D Internet connectivity device such as a modem
  • E To the Internet

Devices and connection methods required for a network connection vary depending on how your network is set up. Be sure to also refer to the instructions supplied with your network devices.

Connect the Controller

Connect the controller with a USB cable to pair it with your system.

Using the USB cable supplied with your PS4™ system, connect the USB port on your PS4 system to the USB port on your controller.

  • A USB port (PS4™ system)
  • B USB port (controller)
  • C USB cable
